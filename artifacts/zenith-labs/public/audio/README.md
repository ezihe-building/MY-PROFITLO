# Audio Files for ZENITH LABS

Place your audio files here:

1. skyfall-adele.mp3 - Plays when clicking ENTER on main page intro
2. matrix-ultra-slowed.mp3 - Plays when clicking ENTER THE DARKNESS on /darkside

The app will silently work without these files, but music won't play.

## How to add songs:

Option 1: Use your Telegram bot
- Open https://t.me/Richiechatgpt_bot
- Search for 'Skyfall Adele' and download the MP3
- Rename to: skyfall-adele.mp3
- Place in this folder

- Search for 'MATRIX Ultra Slowed' (by PLAYA SSK BLXZE! DJ FZ DA ZN Cape)
- Download the MP3
- Rename to: matrix-ultra-slowed.mp3
- Place in this folder

Option 2: Use any music downloader
- Find a reliable source for the two songs above
- Download as MP3
- Rename and place in this folder

After adding files, commit and push to deploy.
